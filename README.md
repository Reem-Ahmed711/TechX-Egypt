## Overview

TechX Egypt is a fully responsive e-commerce web application for electronics, featuring product browsing, filtering, sorting, cart management, user authentication, and an admin dashboard.

## Features

- Product catalog with images, prices, and ratings
- Category filtering and price sorting
- Search functionality
- Shopping cart with quantity management
- User authentication (login/signup)
- Admin dashboard with product CRUD operations
- User management
- Responsive design for all devices

## Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, animations)
- JavaScript (ES6+)
- LocalStorage for data persistence
