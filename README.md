## 🎯 Features in Detail

### For Customers
- ✅ Browse products with images and specs
- ✅ Filter by category (Mobiles, Laptops, Accessories, Gaming)
- ✅ Sort by price (Low to High / High to Low)
- ✅ Search products by name or category
- ✅ Add/remove items from cart
- ✅ Adjust product quantities in cart
- ✅ Create account and login

### For Admin
- ✅ View dashboard with KPIs (Total Products, Users, Rating)
- ✅ Add new products with images and description
- ✅ Edit existing products
- ✅ Delete products
- ✅ View all registered users
- ✅ Manage inventory
